## Deskripsi Proyek
**SmartSafe** adalah proyek simulasi brankas pintar sederhana yang kami rancang untuk tugas UAS kami, mengenai Digital Logic Design dan Embedded System. Proyek ini lahir dari ide kelompok kami yang ingin menguji dan mengimplementasikan pemahaman dasar kami seputar sistem tertanam (*embedded system*) dalam sebuah brankas sederhana.

Disini kami memanfaatkan komponen-komponen *hardware* yang mudah ditemukan di pasaran, seperti Arduino Uno, buzzer (sebagai alarm), dan micro servo (sebagai penggerak kunci brankas). Untuk merealisasikannya secara digital tanpa harus merakit komponen fisik terlebih dahulu, kami memanfaatkan platform **Tinkercad** sebagai media simulasi dan pemrograman sistemnya.

## Cara Menggunakan SmartSafe di Tinkercad
  1. Silahkan buka file **src** pada main-branch.
  2. Copy-Paste link yang ada pada kolom _code_ tersebut dan buka di laman website baru internet anda.
  3. Ketika anda sudah masuk **Tinkercad**, anda bisa langsung jalankan _SmartSafe_ tersebut dengan
     menekan tombol _Start Simulation_ pada bagian pojok kanan atas layar anda.
  4. Silahkan masukkan _password_ yang benar (1234) dengan menekan tombol-tombol angka pada _num-pad_.
  5. Lalu tekan tombol Pagar (#) pada _num-pad_, ini bekerja sebagai tombol _Enter_ dari _SmartSafe_ ini.
  6. Jika anda salah memasukkan _password_, maka akan muncul tulisan **INCORRECT**. Jika anda tidak memasukkan
     _password_ dan menekan berulang-kali secara cepat tombol Pagar (#) atau _Enter_ pada num-pad, maka
     akan ditampilkan tulisan **SYSTEM LOCKED** dengan suara _buzzing_ dan LED berkedap-kedip.
